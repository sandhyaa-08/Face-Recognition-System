   Real-Time Face Recognition System

   real-time face recognition system using Python, OpenCV, and DeepFace.

  Features

- Real-time face detection using webcam
- Face matching with stored images
- Displays person name on screen
- Green rectangle around detected face
- Unknown face detection

 Technologies Used

- Python
- OpenCV
- DeepFace
- TensorFlow

  Folder Structure

Face-Recognition-System/
│
├── faces/
├── main.py
└── README.md

Installation

   Install required libraries:

pip install -r requirements.txt

   Run Project

python main.py


 How It Works

1. Store known face images inside `faces/` folder
2. Run the program
3. Webcam starts automatically
4. System compares live face with stored images
5. If matched, name is displayed
